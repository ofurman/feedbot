# <p align="center">w8feedbot
Bot for Telegram App. This bot forwards messages from channels you follow. 

## Getting Started
Open telegram and search for @w8feedbot. Then open dialog and push `start`. To follow any channel just forward any message from this channel to the bot.

## Available commands:

|name|Condition|
|:---:| ---|
|`/start`|Starts the bot and adds you to database|
|`/mychannels`|Prints a list of channels you follow|
|`/unsubscribe`|Prints a list of channels to unsubscribe|

## Built With

* [flask](https://github.com/pallets/flask) - The web framework used
* [pyTelegramBotAPI](https://github.com/eternnoir/pyTelegramBotAPI) - Python implementation for the <a href="https://core.telegram.org/bots/api">Telegram Bot API</a>
